# DESBLOQUEIO-DE-USU-RIO-NO-ACTIVE-DIRECTORY
DESBLOQUEIO DE USUÁRIO NO ACTIVE DIRECTORY
Este fluxo realiza o desbloqueio de usuário da Rede Microsoft.

Sobre este fluxo de automação:
Este fluxo faz o desbloqueio do usuário na rede Microsoft validando o seguinte fluxo:

Usuario Existe?
Está Bloqueado
Caso estas condições sejam reais ele procede pra o desbloqueio do usuário de rede.
